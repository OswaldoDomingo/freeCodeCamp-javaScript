Elimina propiedades en un objeto de JavaScript

Ejercicio 90

También podemos eliminar propiedades de objetos de esta forma:
delete ourDog.bark;
Por ejemplo: 
        const ourDog = {
            "name": "Camper",
            "legs": 4,
            "tails": 1,
            "friends": ["everything!"],
            "bark": "bow-wow"
          };
          
          delete ourDog.bark;

Después de la última línea mostrada anteriormente, ourDog se ve así: 
        {
            "name": "Camper",
            "legs": 4,
            "tails": 1,
            "friends": ["everything!"]
          }
Ejercicio:
Elimina la propiedad tails de myDog. Puedes usar tanto la notación de puntos como la notación de corchetes. 
    Código original.
    // Configuración
    const myDog = {
        "name": "Happy Coder",
        "legs": 4,
        "tails": 1,
        "friends": ["freeCodeCamp Campers"],
        "bark": "woof"
    };

// Cambia solo el código debajo de esta línea
    
    Código resuelto

    const myDog = {
        "name": "Happy Coder",
        "legs": 4,
        "tails": 1,
        "friends": ["freeCodeCamp Campers"],
        "bark": "woof"
    };
    
    delete myDog.tails;
